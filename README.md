# 🚀 MyHost Infrastructure (Ansible + Incus + Btrfs)

This repository contains the complete automation for my **MyHost** server. The project is designed to manage both the host layer (system) and container orchestration via Incus.

## 🛠 Tech Stack

* **OS:** Debian 13 - Trixie (2026))
* **Kernel:** 6.12+
* **Filesystem:** Btrfs (RAID1 / Subvolumes `@trixie` & `@home`)
* **Virtualization:** [Incus](https://linuxcontainers.org/incus/)
* **Automation:** Ansible
